What Makes a Fight Song “Fight”?

Big Ten fight songs compared

Created by: Srividya Ramanathan, University of Iowa

Overview and Dataset

I wanted to understand how fight songs differ across schools and across time by comparing their structure, language, and rhythm. This project includes two visualizations. One focuses on musical composition. The other focuses on language use over time.

Data source: FiveThirtyEight Fight Songs dataset. This entry focuses on Big Ten institutions.

How to read the main visualization

  • Each school is one slice of the circle, ordered by tempo (BPM).
  • Color shows the factor: Identity (blue), Chantiness (gray), Aggressiveness (black).
  • Spoke length shows level: inner ring is absent, middle ring is noticeable, outer ring is dominant.
  • Inner squiggles encode tempo groups using Italian tempo terms.
  • Hover over any spoke to see the full university name, song title, and BPM.

Fight Song Composition

This interactive volvelle-style visualization compares Big Ten fight songs across three compositional factors: Identity, Chantiness, and Aggressiveness.

Note: “Chantiness” is a term I use to describe how easy a song is to chant in a crowd.

Data Analysis

The volvelle makes it easy to compare patterns across schools without ranking them. It shows that fight songs can signal school spirit in different ways. Some songs rely more on identity and chant structure. Others use more confrontational language. Tempo adds another layer. Faster songs often feel more energetic, but speed does not always match lyrical aggression.

The Word “Fight” Over Time

The second visualization looks at language use over time. It focuses on how often fight songs use the word “fight,” and how this changes across years.

How to read the second visualization

  • Each point is one song.
  • The x-axis is year.
  • Point size shows how many times the word “fight” appears in the lyrics.
  • Color shows whether usage is absent, noticeable, or dominant.

Data Analysis

The timeline view adds context to the compositional comparison. It shows that explicit use of the word “fight” is not constant across eras. Some periods use direct language more often, while others lean more on identity and tradition without repeating the word itself.
